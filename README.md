# J-Code Theme for VS Code

![J-Code Theme Preview](.github/v1.1.1.png)

O **J-Code** é um tema escuro que criei para facilitar meu próprio uso diário no Visual Studio Code.
Depois de testar vários temas, resolvi ajustar um que fosse mais previsível, confortável e fácil de ler ao longo do tempo.

Ele não tenta chamar atenção — a ideia é deixar o código falar por si.

---

## Por que esse tema existe

O objetivo foi simples:

- Ler código com menos esforço
- Manter uma hierarquia visual clara
- Evitar cores excessivas ou estilos chamativos
- Funcionar bem em diferentes linguagens

O J-Code surgiu de ajustes práticos, feitos conforme o uso real.

---

## Características

- **Paleta contida**
  Apenas as cores necessárias para diferenciar estruturas importantes.

- **Hierarquia semântica consistente**:
  - Tipos, interfaces e enums em teal
  - Classes em uma variação mais escura
  - Funções e métodos em azul
  - Palavras-chave em roxo suave
  - Strings em amarelo
  - Números e constantes em laranja
  - Variáveis no foreground principal
  - Comentários sempre discretos

- **Semantic Highlighting ativado**
- **JSX/TSX legível**, com texto dentro de tags claramente distinto de comentários
- **Ajustado para JetBrains Mono**, sem exagero de bold ou itálico

- Menos distração

Não é um tema universal — é um tema que funciona bem para mim e pode ser útil para outras pessoas.

## Instalação

1. Abra o Visual Studio Code
2. Extensions (`Ctrl+Shift+X`)
3. Procure por **J-Code**
4. Ative em `File > Preferences > Color Theme > J-Code`

Sugestão: usar com a fonte **JetBrains Mono**.
